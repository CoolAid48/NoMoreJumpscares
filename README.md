### Removes the Elder Guardian curse effect when receiving mining fatigue!

## This mod features:
- **Disabled** Elder Guardian curse particle popup *and* curse sound
- This mod **only** disables the jumpscare, **not** the mining fatigue effect
- Purely *client-side*, **no server-side functionality**
